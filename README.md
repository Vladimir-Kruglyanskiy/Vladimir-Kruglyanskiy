## Почему аналитика данных?

Работая в банке с большими объёмами информации, я столкнулся с ограничениями Excel: медленная обработка данных, невозможность автоматизации. Я начал искать решения и пришёл к SQL (стандарт работы с базами) и Python (гибкость, статистика, визуализация). 

Чтобы получить системные знания, прошёл специализированные курсы. Так мой интерес перерос в осознанный выбор профессии аналитика данных.

**Мой опыт в фарминдустрии и банковской сфере** помогает мне формулировать гипотезы с опорой на реальное поведение и потребности пользователей.
